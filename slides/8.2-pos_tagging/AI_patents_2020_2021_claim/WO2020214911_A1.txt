<title>
METHOD AND SYSTEM FOR GENERATING ATTENUATION MAP FROM SPECT EMISSION DATA BASED UPON DEEP LEARNING
</title>

<abstract>
A system for estimating attenuation coefficients from only single photon emission computed tomography (SPECT) emission data using deep neural networks includes an artificial neural network based upon machine learning system estimating attenuation maps for SPECT emission data, and associated attenuation correction method.
</abstract>

<claims>
1. A system for estimating attenuation coefficients or attenuation maps (ATTMAP) from only single photon emission computed tomography (SPECT) emission data using artificial neural networks, comprising:
a machine learning system based upon deep artificial neural networks for estimating attenuation maps for SPECT emission data.
2. The system according to claim 1 , wherein the machine learning system includes a generator network estimating attenuation maps for SPECT emission data and a discriminator network enforcing output of the generator network to be consistent with a ground truth attenuation map.
3. The system according to claim 2, wherein the generator network is trained.
4. The system according to claim 3, wherein the generator network is trained with Generative Adversarial Network training.
5. The system according to claim 3, wherein the generator network is trained with an Adam optimizer.
6. The system according to claim 2, wherein the discriminator network is trained.
7. The system according to claim 6, wherein the discriminator network is trained with an Adam optimizer.
8. The system according to claim 2, wherein the generator network is a deep convolutional neural network.
9. The system according to claim 8, wherein the discriminator network is a deep convolutional neural network.
10. The system according to claim 2, wherein the discriminator network is a deep convolutional neural network.
11. The system according to claim 1 , wherein the ground truth attenuation map is generated based upon empirical evidence.
12. The system according to claim 1 , wherein the SPECT emission data includes images reconstructed from a photopeak window and one or more scatter windows.
13. The system according to claim 12, wherein the images reconstructed from the photopeak window and the scatter window are concatenated as a multi-channel image and fed into a generator network.
14. The system according to claim 1 , wherein the SPECT emission data includes images reconstructed from a photopeak window.
15. The system according to claim 1 , wherein the SPECT emission data includes images reconstructed from one or more scatter windows.
16. A method for generating attenuation maps and performing associated attenuation correction from SPECT emission data, comprising:
generating an attenuation map from a NAC SPECT image dataset through deep learning;
estimating attenuated projection data via forward projecting the NAC SPECT image without the attenuation map; and
reconstructing an AC SPECT image from the estimated attenuated projection data using iterative reconstruction with attenuation correction by incorporating the attenuation map generated by deep learning.
17. The method according to claim 16, where the step of generating is performed using a machine learning system based upon artificial neural networks.
18. The method according to claim 17, wherein the artificial neural network includes a generator network.
19. The method according to claim 18, wherein the generator network is a deep convolutional neural network.
</claims>

<title>
SYSTEMS AND METHODS FOR PLAYBACK RESPONSIVE ADVERTISEMENTS AND PURCHASE TRANSACTIONS
</title>

<abstract>
In a method for displaying tailored advertising in response to user media playback behavior, an ad engine detects media playback by a user. The ad engine analyzes content associated with the media playback and queues user tailored advertisements based on the analyzed content. The ad engine further displays an interactive element associated with the user tailored advertisements. Upon receiving a user interaction with the interactive element, the ad engine modifies the delivery of the user tailored advertisements based on the user interaction.
</abstract>

<claims>
1. A method for controlling displayed advertisements in response to a user action via an over-the-top data delivery system, comprising: detecting media playback by the user; analyzing content associated with the media playback; queueing user tailored advertisements based on the analyzed content; displaying an interactive element associated with the user tailored advertisements; receiving a user interaction with the interactive element; and modifying the delivery of the user tailored advertisements based on the user interaction.
2. The method of claim 1, further comprising queueing the user tailored advertisements based on an advertisement rule associated with the user.
3. The method of claim 2, wherein the advertisement rule controls selection of advertisements based on a variable selected from the group consisting of: permitted brands, permitted themes, advertisement length, advertisement content, advertisement format, seasonality, user advertisement preferences, derived user advertisement preferences, and technical specifications of a playback device.
4. The method of claim 1, wherein analyzed content associated with media playback further comprises analyzing a media attribute selected from the group consisting of: total playback time, format, designated commercial breaks, subject matter, themes, actors, production studios, and playback permissions.
5. The method of claim 1, further comprising: modifying an advertisement rule based on one or more user-inputted advertisement parameters associated with the user interaction with the interactive element; and applying the modified advertisement rule to a group of available advertisements.
6. The method of claim 1, wherein the advertisement is configured to receive user input via a user interface.
7. The method of claim 6, further comprising: receiving a user purchase request associated with the advertisement; executing the user purchase request; and causing one or more of a product and a service to be delivered to the user.
8. The method of claim 5, wherein the interactive element is associated with provides additional information to the user regarding the user-tailored advertisements.
9. The method of claim 5, wherein the interactive element is associated with provides additional information to the user regarding the analyzed content.
10. The method of claim 9, wherein the additional information is sourced from a remote third-party data store comprising one or more collective databases coupled to at least a first entity and a second entity.
</claims>
